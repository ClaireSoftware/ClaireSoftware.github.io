Nós temos salas de chat (baseadas em IRCm na rede freenoda), que podem ser ser usadas por membrxs da comunidade para se contactarem entre si:

O flag +s está definido em todos estes canais, para que ninguém saiba que você está nestes canais quando estes usam o comando whois no seu nick de IRC. A única forma de alguém descobrir quem você é é juntar-se a um dos nossos canais e vê-lx as si lá.

Estes são espaços seguros. Você é livre de se expimir aqui, sem ser julgadx.

  • ##Libretransit Este é um canal de apoio, para pessoas que necessitam de apoio. Não é um canal para socialização. Para discutir assuntos em geral ou para socialização (incluindo mesmo coisas que não fazem parte dos tópicos de um grupo de apoio trans), vá ao canal ##Libretransit-social (veja o link abaixo):
  • ##Libretransit-social Este é um local de encontro para a comunidade, para pessoas de todas identidades de género (mulheres, homens, pessoas não binárias, etc). Qualquer pessoa é bem-vinda. Pessoas intersexo e eunucxs também são bem-vindxs. Este é um local para socialização. Apoio/suporte pertence no canal principal!)
  • ##Libretransit-nsfw - tópicos delicados pertencem a este canal.

Os links acima são chats baseados na web, mas vocês também podem usar outros clientes de IRC em GNU+Linux, BSD, MacOS, Windows, Android, Replicant, iOS etc (GNU+Linuxm, BDS ou Replicant são bastante recomendados, para proteger a vossa liberdade contra a tirania como utilizadorxs de computadores.

As informações para se ligar ao IRC são as seguintes:

  • Servidor: chat.freenode.net
  • Porta: 6697
  • Encriptação TLS/SSL é necessária nesta porta. Você tem que ativar encriptação
  • Os nossos canais são ##Libretransit, ##Libretransit-nsfw e ##Libretransit-meta (certifique-se que usa dois “##” em vez de um apenas.

Quer você esteja a usar um web chat, ou um cliente de IRC não baseado em web, é altamente recomendável que você registe o seu nome de IRC. Escreva o seguinte comando no seu cliente de IRC para informação sobre como registar o seu nome de IRC:

/msg nickserv help register
