English – Português

Para esconder o seu pénis, você pode construir uma gaff usar fita adesiva.

Se você estiver a usar fita adesiva, então empurre os seus testículos para dentro dos seu corpo (para dentro dos seus canais inguinais) e puxe o seu pénis para baixo e para o meio das suas pernas, para que este aponte para longe do seu corpo atrás de si. Depois cole a fita adesiva. Você deve rapar os pêlos rents à pele na área dos seus genitais antes de usar a fita adesiva. PLANOS: arranje uma marca/tipo de fita adesiva adequados.

Para conforto, segurança e conveniência nós recomendamos usar uma gaff.

Há muitos guias sobre tucking. Use o Google e/ou YouTube (ou talvez outros sites) para encontrar o tipo de técnica que você quer usar.

Você também pode usar 2 ou 3 pares de cuecas apertadas.

Para remover o seu pénis, leia esta página.
