This is a (poorly designed) tutorial about how to use Adafruit's PiOLED With a Raspberry Pi Zero to make a portable thumbdrive with Kali Linux.

What You'll Need

  • A Raspberry Pi Zero WH (Unfortunately, it's hard to avoid paying the adafruit tax on this one.
  • A Adafruit PiOLED A pretty nice oled for the price, and not too overpriced.
  • A Pi Zero Badusb Case A really nice case for the pi zero, compatible with the pi zero. Lets you power it off of a standard USB port. You could use a microUSB instead.
  • A MicroSD card. 16GB or so should be fine. Try for class 10 :)

How to Setup the Pi

  • Image Raspbian Lite to the SD card, turn it into an ethernet gadget. (Put dtoverlay=dwc2 at the bottom of config.txt, put modules-load=dwc2,g_ether in b between rootwait and quiet in cmdlline.txt). Add another user and give them sudo access, or simply use root all the time like kali does.
  • Run the following script on the pi as administrator to install mate and katoolin. (Credit to jerry gamblin for original script :) ) Katoolin seems to want to pull from an invalid keyserver, so I made a small modification to make it pull from a valid one.
    #!/bin/bash
    apt-get update && sudo apt dist upgrade -y
    apt-get -y install --no-install-recommends git mate-core mate-desktop
    apt-get install mate-core mate-desktop-environment mate-notification-daemon xrdp iceweasel xserver-xorg-video-fbdev -y
    echo mate-session> ~/.xession
    cp ~/.xsession /etc/skel
    git clone https://github.com/ClaireSoftware/katoolin.git && cp katoolin/katoolin.py /usr/bin/katoolin
    chmod +x /usr/bin/katoolin
  • Use raspi-config to configure I2C in interfacing options, and then add your user to the i2c group.
  • Install the needed libraries for I2C, and run the example program.
    sudo apt install build-essential python-dev python-pip python-imaging python-smbus
    sudo pip install RPi.GPiO
    git clone https://github.com/adafruit/Adafruit_Python_SSD1306.git
    cd Adafruit_Python_SSD1306
    sudo python setup.py install
  • Test the OLED :)
    cd examples
    sudo python stats.py # Needs to be run as sudo until reboot

Done!
